¿Qué software hemos usado?

Hemos usado varios programas o aplicaciones para realizar este trabajo, como por ejemplo el MIT App Inventor, Arduino IDE, Fritzing, AutoCAD y nuestra propia aplicación para poder conectarnos a nuestro modulo bluetooth (HC-06), ademas si haces click en el nombre de la aplicación o programa, este te llevará a la web de descarga. A continuación explicaremos cada uno de ellos:

  1. Arduino IDE: Arduino IDE nos proorciona un entorno de trabajo en el que podemos programar en C, con el fin pasar el código a la placa y que funcione haciendo lo que hemos programado, Arduino IDE está disponible para Windows, Linux y Mac.

  2. MIT App Inventor: MIT App Inventor nos proporciona un lugar de trabajo online en el que podemos crear aplicaciones basicas para android, este consta de dos modos (diseñador y de bloques).

  3. Fritzing Fritzing es un programa libre de automatización de diseño electrónico que busca ayudar a diseñadores y artistas para que puedan pasar de prototipos (usando, por ejemplo, placas de pruebas) a productos finales. Fritzing fue creado bajo los principios de Processing y Arduino, y permite a los diseñadores, artistas, investigadores y aficionados documentar sus prototipos basados en Arduino y crear esquemas de circuitos impresos para su posterior fabricación. Además, cuenta con un sitio web complementario que ayuda a compartir y discutir bosquejos y experiencias y a reducir los costos de fabricación.

  4. AutoCAD:Con este programa podemos crear planos tanto en 2D como en 3D, este programa tiene una caracteristica destacable y es que se pueden introducir comandos en el mismo programa pudiendo hacer, justo lo mismo de manera gráfica que por línea de comandos, por decirlo así.

  5. Secure House: Es la aplicación que mediante Bluetooth se puede controlar los componentes de la casa.
