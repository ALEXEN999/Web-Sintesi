Que componentes hemos usado?

Hemos usado una gran variedad de componentes para realizar este trabajo, a continuación mostraremos los componentes y como hemos usado cada uno, ademas al hacer click en el nombre de los componentes, te llevará a una web de compra para obtener el producto:

  1. Leds(transparentes, rojos, azules, verdes, amarillos): Estos leds los usamos para la iluminación de la casa, las luces del coche fantastico, las luces de los sensores de movimiento y Ultrasonido.

  2. Módulo Bluetooth (HC-06): Este módulo bluetooth lo hemos usado para controlar todas las luces desde nuestra applicación creada con MIT inventor.

  3. Buzzer activo: Hemos usado el buzzer activo junto al sensor Ultrasonic para que cada vez que se active el sensor el buzzer realice un pitido junto a un led que se encenderá, en función de cuanto tiempo esté activo.

  4. ultrasonic sensor (HC-SR04): Hemos usado el ultrasonic sensor junto al buzzer activo para que cada vez que se active el sensor el buzzer realice un pitido junto a un led que se encenderá, en función de cuanto tiempo esté activo.

  5. Resistencias: Usamos las Resistencias todo el rato para los leds y el modulo bluetooth, para limitar el valor de la corriente o para fijar el valor de la tensión.

  6. Sensor PIR: Este es un sensor de movimiento que lo tenemos conectado con un led de forma que al detectar movimiento, el led se encenderá y se apagará cuando deje de registrar movimiento.

  7. Protoboard: La Protoboard la usaremos para poder hacer el circuito, como dice el nombre este es una tabla de pruebas, donde podremos probar de hacer circuitos, como en nuestro caso con una placa arduino, y es necesario unos cables de macho a macho, macho a hembra o hembra hembra, depende del circuito que se vaya a hacer.

  8. Placa Arduino: Con la placa Arduino, en nuestro caso Arduino Mega, podemos hacerun montón de cosas, y es la que se encarga de que todo funcione gracias a un codigo programado en "C" que se sube a la placa mediante la aplicación de Arduino IDE, y esta conectada por los cables a la protoboard con el resto de componentes.
