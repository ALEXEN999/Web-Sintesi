Que componentes hemos usado?

Hemos usado una gran variedad de componentes para realizar este trabajo, a continuación mostraremos los componentes y como hemos usado cada uno:

  1. Leds(transparentes, rojos, azules, verdes, amarillos): Estos leds los usamos para la iluminación de la casa, las luces del coche fantastico, las luces de los sensores de movimiento y Ultrasonido.

  2. Módulo Bluetooth (HC-06): Este módulo bluetooth lo hemos usado para controlar todas las luces desde nuestra applicación creada con MIT inventor.

  3. Buzzer activo: Hemos usado el buzzer activo junto al sensor Ultrasonic para que cada vez que se active el sensor el buzzer realice un pitido junto a un led que se encenderá, en función de cuanto tiempo esté activo.

  4. ultrasonic sensor (HC-SR04): Hemos usado el ultrasonic sensor junto al buzzer activo para que cada vez que se active el sensor el buzzer realice un pitido junto a un led que se encenderá, en función de cuanto tiempo esté activo.
